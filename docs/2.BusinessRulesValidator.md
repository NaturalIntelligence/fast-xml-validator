
If your XML has no syntax issue and can be parsed through Fast-XML-Parser to JS object without any error then you can use Business rule validator for schema validation.

```js
const {BusinessRulesValidator} = require("fast-xml-validator");
const validator = new BusinessRulesValidator(rules);
const failures = validator.validate(xmlData, {
    allowBooleanAttributes: true
});
```

Please see the detailed documentation at [detailed-xml-validator](https://github.com/NaturalIntelligence/detailed-xml-validator).
